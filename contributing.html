1. Overview

We welcome contributions to improve the 3D Printer Cost Calculator — from core features to UI enhancements, testing, and documentation.

2. Environment Setup

2.1. Backend (.NET Core)

  • Install .NET 6.0+

  • Use Visual Studio, Rider, or VS Code

2.2. Frontend (Angular)

  • Install Node.js + npm

  • Angular CLI required

3. Directory Structure

  • src/ – ASP.NET Core backend

  • ClientApp/ – Angular frontend

  • docs/specs/ – AsciiDoc specs

  • .github/workflows/ – CI/CD automation

4. Running Tests

4.1. Backend

dotnet test

4.2. Frontend

cd ClientApp
ng test

5. Git Flow

Use these branches: - main: production - develop: integration - feature/xyz, bugfix/abc, docs/…​

6. Pull Request Checklist

  • ❏ Code compiles and passes tests

  • ❏ PR is based on develop

  • ❏ Description is complete

  • ❏ Linked to issue(s)

7. Automation

GitHub Actions automate: - Spec publishing - Issue label tracking - CI pipeline

8. Need Help?

Open a discussion on GitHub or comment on an issue. Contributions are welcome at all levels.
